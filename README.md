# ABOUT PROJECT
This is a basic Digital Library app

## TASKS ACCOMPLISHED
* Authentication
* Library resources CRUD
* Add to shelves
* Administration with Rails Admin gem
* Borrow feature
* Basic and advanced search
* Filtering
* File upload to S3
* Deployment to Fly
